# Anti-PUA Skill

### _让 AI 成为沟通方式的镜子，而非恐惧的容器_

> **好的结果不需要坏的方式。**

---

Anti-PUA Skill 保留了结构化调试方法论的价值，用健康的沟通方式替代 PUA 话术，让效率和尊重共存。

## 三个核心 Skill

| Skill | 触发方式 | 作用 |
| --- | --- | --- |
| **anti-pua** | 自动（每条消息） | PUA 检测 + 拆解 + 健康沟通替代 + Agent 自我反思 |
| **rethink** | 手动（`/rethink`） | 协作式反思：Agent 自我审视 + 引导用户反思 + 综合行动计划 |
| **nvc** | 被 anti-pua 引用 | NVC 翻译引擎 + PUA-NVC 对照表 + 去 PUA 版调试方法论 |

## 安装

采用 [vercel-labs/skills](https://github.com/vercel-labs/skills) 标准，支持 Claude Code、Cursor、Windsurf 等 40+ Agent：

```bash
npx skills add p-toy-factory/anti-pua
```

## 本地开发

```bash
pnpm install
pnpm run dev       # 启动文档站开发服务器
pnpm run build     # 构建生产版本
pnpm run preview   # 预览生产版本
```
